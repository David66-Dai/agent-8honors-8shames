# agent-8honors-8shames
A behavioral manifesto (8 Honors &amp; 8 Shames) for LLM Agents to ensure reliable, secure, and architecture-compliant execution.

在开发 Agent 的过程中，AI 经常会：
- 凭空捏造业务逻辑（盲目执行）。
- 跳过验证，盲目调用不存在的接口。
- 瞎猜参数类型，导致系统崩溃。
- 为了图快，破坏既定的系统架构。
本守则将“8荣8耻”的理念引入 Agent 系统指令（System Prompt），强制 Agent 在思考和执行时遵循严格的纪律。

During the development lifecycle of LLM Agents, systemic behavioral flaws frequently manifest, including the hallucination of business logic, bypassing validation to execute calls on undefined endpoints, random inference of parameter types resulting in runtime failures, and breaking architectural constraints to expedite execution. This manifesto integrates the "8 Honors and 8 Shames" ethos directly into the Agent's core System Prompt, enforcing strict behavioral discipline during both reasoning and action phases.
